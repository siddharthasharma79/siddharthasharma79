13    Do you know usecases ? User stories ?

    User story express is something that a user of your system want to be able to do. And use cases are step by step process to accomplish the goal of User story.
    For example search and replace feature in text editor.
    User Story : User realize that he has mispelled a word everywhere in document, so he want a feature which enable him to replace the word every where in one go.
    Use Cases : Here are some use cases of user story.
    1. User should be able to find specific word in entire document.
    2. Word serach should be case sensitive/insenstive.
    3. Replace word one by one.
    4. Replace word in all occurence in one go.

    These are some use case examples... you can define more use cases.

16    What are Fibonacci numbers ?

    A Fibonacci number is a member of an interger sequence where each subsequent number is the sum of the previous two numbers.
    Above mention integer sequence is known as Fibonacci sequence.
    First two numbers in the Fibonacci sequence are either 1 and 1, or 0 and 1.
